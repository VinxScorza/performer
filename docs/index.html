PER|FORMER

eurorack sequencer

forked and maintained by Vinx Scorza

PER|FORMER is a eurorack sequencer module designed for both live performance and work in the studio. This Vinx Scorza fork is a personal experimental line shaped by real performance use, custom behavior, and focused UI/interaction changes.

GitHub fork: github.com/VinxScorza/performer

Documentation note: the User Manual is now aligned through v0.3.2-vinx.1.4.8, including the appendix and Launchpad sections.

Lineage: derived from the Mebitek fork of Westlicht Performer.

IMPORTANT NOTE: I am not a software developer. I am just an artist shaping the instrument I want to perform with.

What’s New in Vinx Fork

Current fork line: v0.3.2-vinx.1.4.8
This fork remains intentionally personal and exploratory, shaped by ongoing use of PER|FORMER as a live instrument.

Fixes & Stability

  • Fix crashes/reboots on 64-step Note sequence shifting and correct step-shift range handling across all sequence types (Note, Logic, Curve, Stochastic, Arp)
  • Restore proper Init behavior (Init Layer vs Init Steps) and resolve generator/system/UI edge cases, including dead code paths

Chaos

  • Introduce experimental CHAOS MODE with Vandalize Sequence and pattern-wide Wreck Pattern across all Note tracks
  • Refine workflow: enter on ORIGINAL, explicit CHAOS trigger, A/B preview, and VANDALIZED / WRECKED popups
  • Improve behavior: per-track First Step/Last Step ranges, consistent preview triggering, and warning popup before Wreck Pattern (F1 PROJ PAGE, F3 WRECK, F5 cancel)
  • Add System -> Chaos Defaults with persistent 4x4 layer masks for sequence/pattern targets, auto-saved to flash
  • Optimize memory: target-step backups only plus in-place preview regeneration to reduce runtime pressure

WARNING: Chaos -> Wreck Pattern is a wildly experimental, pattern-wide process and should be used only after saving your project.

Generators

  • Add the new Acid Note-track generator with Layer / Phrase modes, where Phrase writes coordinated Gate + Note + Slide material
  • Add experimental Chaos as a multi-layer macro-random generator with Amount, preview, and matrix-based target control
  • Rework Euclidean (A/B, NEW EUCL, direct Steps/Beats/Offset, Beats <= Steps) and Random (A/B preview, Variation, 32-bit hex seeds, auto reseed, layer-aware rendering)
  • Make RESETGEN restore true defaults, compact generator selection to a single page, and redesign LCD previews for Random and Acid

Workflow, UI & Interaction

  • Refine step visualization (compact conditions, retrig in-step, slide ties)
  • Keep transport always responsive during generator usage
  • Add Scale prelisten on Note/Arp with Cancel support and cyclic sequence navigation
  • Improve contextual labeling (ACID PHRASE, CHAOS ON SEQ/PAT, etc.)

System & Performance

  • Reduce memory footprint and allocations, especially around Chaos; lower UI refresh to 30 fps; skip redundant LCD redraws via framebuffer hashing
  • Increase default clock pulse (1 ms to 10 ms), add three Dim Sequence levels, and set screensaver to 15m with improved wake behavior
  • Make the System page land on Settings by default and align startup/version reporting with v0.3.2-vinx.1.4.8

Content & Ecosystem

  • Restore browser Test Drive, update the manual to the Vinx fork line, publish the Launchpad cheat sheet, and refresh simulator/demo assets
