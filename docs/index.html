PER|FORMER

eurorack sequencer

forked and maintained by Vinx Scorza

PER|FORMER is a eurorack sequencer module designed for both live performance and work in the studio. This Vinx Scorza fork stays close to the original instrument while refining workflow details and adding focused usability changes for personal use. Personal experimental fork focused on live workflow, custom behavior, and UI/interaction changes.

GitHub fork: github.com/VinxScorza/performer

Documentation note: the User Manual is now aligned through v0.3.2-vinx.1.4.7, including the appendix and Launchpad sections.

Lineage: Mebitek fork of Westlicht Performer.

IMPORTANT NOTE: I am not a developer. I am an artist trying to shape and carve his own instrument.

About Vinx Scorza: vincenzoscorza.it

What’s New in Vinx Fork

A quick overview of the most relevant fixes and workflow changes introduced in the Vinx fork beyond Mebitek v0.3.2.

Fixes

  • Fix crashes and reboots when shifting 64-step Note sequences
  • Fix step-shift range handling across Note, Logic, Curve, Stochastic, and Arp sequences
  • Restore the browser-based Test Drive as a working public build of the current fork line
  • Include further small UI, workflow, and dead-code fixes beyond the headline changes

Improvements

  • Add the new Acid Note-track generator with Layer / Phrase modes, where Phrase writes coordinated Gate + Note + Slide material
  • Add the new experimental Chaos Note-track generator: a multi-layer macro-random tool with non-destructive preview, global Amount, direct All On / All Off actions, and a dedicated layer matrix
  • Refine step visualization again with compact condition marks, retrigger marks moved into the step box outside the dedicated Retrig layer, and a small slide tie between adjacent steps
  • Fix the Init Layer regression and clarify step initialization: SHIFT + PAGE -> INIT LAYER now resets only the active layer again, while GEN -> Init Seq resets sequence steps with the same selection-aware scope used by the other generators
  • Change the default screensaver timeout to 15m and make the first gesture after wake restore only the display when Wake Mode is set to required
  • Add non-destructive Random preview with A/B comparison and Variation
  • Extend Random to 32-bit seeds shown as 8-digit hexadecimal values, randomize the seed on entry and on change, and remove the old linear seed stepping workflow
  • Make Random previews layer-aware on Note tracks
  • Add immediate Scale prelisten on Note and Arp sequence pages
  • Add Cancel while editing Scale and Root Note
  • Redesign generator previews for Random and Acid on the LCD
  • Make sequence-page prev / next navigation cyclic
  • Rescale the Swing display from 50%–75% to 0%–99% while preserving engine timing compatibility with existing projects
  • Introduce three Dim Sequence levels instead of a single on/off behavior
  • Increase the default output clock pulse from 1 ms to 10 ms
  • Lower the default UI refresh to 30 fps
  • Skip LCD redraws when the framebuffer hash has not changed
  • Update and realign the manual to the current Vinx fork line
  • Publish the Launchpad cheat sheet on the site
  • Refresh the built-in simulator demo project, drumkit, and browser sound setup for the current line

See the full CHANGELOG for the complete list of Vinx-specific changes.

Vinx Fork Notes

  • Current fork line: v0.3.2-vinx.1.4.7
  • CHANGELOG tracks Vinx-specific fixes and features
  • README explains the fork status and upstream references
  • This firmware line was built on top of the Mebitek fork because it already provided features I needed; some inherited functionality remains here as part of that lineage even when it is not central to my own daily workflow
  • The fork stays intentionally personal and exploratory, shaped by ongoing use of PER|FORMER as a live instrument
  • Chaos is a deliberately rough multi-layer Note-track generator intended more as controlled vandalism than as a musically “smart” phrase tool
  • Backward compatibility with older projects, settings, or workflows is not guaranteed for Vinx-specific changes

Features

  • 8 track sequencer
  • 16 sequences per track
  • 64 steps per sequence
  • Multiple track and sequence modes
  • Note and modulation sequencing
  • Arpeggiator
  • Snapshot system
  • Sequence generators with non-destructive Random preview workflow
  • Routing system for CV inputs and external MIDI controllers
  • Novation Launchpad support
  • Generate MIDI output from multiple sequences
  • Flexible clock system

User Interface

  • Large and bright 256x64 pixel OLED display
  • Encoder with push button
  • 32 buttons with bi-color LEDs
  • 5 function buttons with labels on display

Inputs and Outputs

  • 8 CV outputs (-5V to 5V)
  • 8 gate outputs (5V)
  • 4 CV inputs (-5V to 5V)
  • Clock and reset/run input
  • Clock and reset/run output
  • MIDI input and output (MIDI TRS 3.5mm jacks)
  • USB Host for MIDI over USB
  • microSD Card Slot

Specifications

  • Width: 34 HP
  • Depth: 25mm
  • Power: 100mA @ +12V, 15mA @ -12V (not including USB host power)
