Detach Reference

Global

Function Notes
TOP 8 + TOP 1..3 Enter Sequence | Pattern | Performer modes. Main Launchpad modes.
TOP 8 SHIFT Main global modifier.
TOP 8 + TOP 7 Toggle PLAY | STOP. Global transport toggle.
TOP 8 + TRK 1..8 Toggle Track mute. Available in Sequence, Pattern, and Performer.
In Sequence Generators Mode, scene buttons are remapped and this action is not available.
TOP 7 + TRK 1..8 Track Fill. Momentary while held. Releasing TRK or TOP 7 clears Fills.
In Sequence Generators Mode, scene buttons are remapped and this action is not available.
TOP 6 + TRK 1..8 Toggle Track solo. Available in Sequence and Performer only.
Not available in Pattern.
In Sequence Generators Mode this action is not available.
TOP 1 + GRID Navigation map. Shows the current mode Navigation map.
Performer Overview uses GRID 25..32 map positions.
In Sequence Generators Mode, GRID is reserved for generator controls.
TOP 1 + TOP 2..5 Navigate Left | Right | Up | Down. Use these arrows on the current mode Navigation map, or press a pad directly.
In Sequence Generators Mode, these arrows are ignored.
Global visual feedback Color indicators in Global. TRK 1..8 LEDs show playback, mute, and selection state.
Colors follow LP Style setting.

Note Circuit Keyboard

This Sequence sub-view is used only when LP Note Style is set to Circuit and the selected Note Layer is Note.

Function Notes
GRID 1..16 Edit current 16-step section. Hold a note pad in 26..39 (Note Keyboard), then press a step pad in 1..16 to write that note and force gate on.
Without a held note pad in 26..39, pressing a step pad in 1..16 toggles gate.
Navigation chooses which 16-step section is shown here.
GRID 26..39 Note keyboard. 26..31 = semitone row.
33..39 = tone row.
Scale-aware note selection.
In-scale notes follow the current scale; out-of-scale notes remain directly selectable chromatically (bypass scale).
GRID 49..56 Octave select. -4 | -3 | -2 | -1 | 0 | +1 | +2 | +3
Selected octave is remembered.
GRID 57..60 Select 16-step bank.

Legacy / Deprecated		 Toggle scale base note at octave 0 | octave +5. Kept for compatibility. Switches the pressed step between the scale base note at octave 0 and the same note at octave +5.

Stochastic Circuit Keyboard

This Sequence sub-view is used only when LP Note Style is set to Circuit and the selected Stochastic Layer is Note Prob.

Function Notes
GRID 1..16 Edit note variation probability for the selected note target. Press a note target pad in 26..40, then press a pad in 1..16 to set its note variation probability from low to high.
If the selected target probability is 0, pads 1..16 stay OFF.
GRID 26..40 Note target select. 26..31 = semitone row.
33..39 = tone row.
32 = raise global Track octave by +1.
40 = lower global Track octave by -1.
Scale-aware note selection.
In-scale notes follow the current scale; out-of-scale notes remain directly selectable chromatically (bypass scale).
Double-press on 26..40 toggles selected note state and stores current octave in the selected stochastic step.
GRID 49..56 Octave select. -4 | -3 | -2 | -1 | 0 | +1 | +2 | +3
Selected octave is remembered.
GRID 57..59 Stochastic options.
57
Use Loop		58
Clear Loop		59
Reseed

Reseed is blocked if Use Loop is enabled or the sequence is empty.

Arp Circuit Keyboard

This Sequence sub-view is used only when LP Note Style is set to Circuit and the selected Arp Layer is Note.

Function Notes
GRID 1..16 Edit gate probability for the selected note target. Press a note target pad in 26..40, then press a pad in 1..16 to set its gate probability from low to high.
If the selected target probability is 0, pads 1..16 stay OFF.
GRID 26..40 Note target select. 26..31 = semitone row.
33..39 = tone row.
32 = raise global Track octave by +1.
40 = lower global Track octave by -1.
Scale-aware note selection.
In-scale notes follow the current scale; out-of-scale notes remain directly selectable chromatically (bypass scale).
When MIDI keyboard mode is ON (57), pressing 26..40 feeds the arpeggiator and releasing non-gated notes removes them.
Double-press on 26..40 toggles selected note state only when MIDI keyboard mode is OFF; when ON, double-press has no action.
GRID 49..56 Octave select. -4 | -3 | -2 | -1 | 0 | +1 | +2 | +3
Selected octave is remembered.
GRID 57..58 MIDI keyboard options.
57
Toggle MIDI keyboard		58
Toggle Hold

Hold works only when MIDI keyboard mode is ON.

Behavior is Hold.

Pattern visual feedback Color indicators in Pattern. GRID colors vary with LP Style.
They distinguish edited patterns, the currently playing pattern, and requested pattern changes.

Sequence Length Layer TOP 2 + GRID 1

Function Notes
GRID Temporary first/last step override across all Note | Curve | Stochastic | Logic Tracks. Hold one pad to loop that single step. Hold two pads to set a temporary first/last range. Releasing the held pad(s) restores the original range.

Overview Layer TOP 2 + GRID 2

Function Notes
GRID Rows address track lanes and columns map visible steps. Toggles gates directly on Note, Stochastic (first 12 only), and Logic.
TRK 1..8 Track select. Scene row selection for the currently selected track.
TOP 1 + GRID 25..32 Performer Navigation. Moves the visible Overview window; other GRID pads are ignored in this map.
TOP 1 + TOP 2..5 Navigate Left | Right | Up | Down. Used on the current Performer Navigation map.
TOP 7 + GRID
Legacy / Deprecated		 Toggle scale base note at octave 0 | octave +5. Kept for compatibility. On Note Track rows, switches the pressed step between the scale base note at octave 0 and the same note at octave +5.
Overview visual feedback Color and bank indicators in Overview. Current step and gates use different LP Style color cues.
When the legacy octave +5 toggle is used, that note has its own cue too.
TOP 1..8 LEDs indicate the current bank of 8 steps for the selected track.
