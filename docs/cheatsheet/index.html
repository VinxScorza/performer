Global

Function Notes

Sequence Sub-View: Note Circuit Keyboard

This sub-view is used only when LP Note Style is set to Circuit and the selected Note Layer is Note.

Function Notes
GRID 1..16 Edit current 16-step section. Hold a note pad in 25..40 (Note Keyboard), then press a step pad in 1..16 to write that note and force gate on.
Without a held note pad in 25..40, pressing a step pad in 1..16 toggles gate.
Navigation chooses which 16-step section is shown here.
GRID 25..40 Note Keyboard. 25..32 = semitone row.
33..40 = tone row.
If the chosen note exists in the current scale, note selection follows the scale.
Notes outside the current scale use a separate chromatic selection.
GRID 49..56 Octave select. -4 | -3 | -2 | -1 | 0 | +1 | +2 | +3
Selected octave is remembered.
GRID 57..60 Select 16-step bank. 57 = steps 1-16
58 = steps 17-32
59 = steps 33-48
60 = steps 49-64
TOP 7 + 1..16 Toggle scale base note at octave 0 | octave +5. Switches the pressed step between the scale base note at octave 0 and the same note at octave +5.

Sequence Sub-View: Stochastic Circuit Keyboard

This sub-view is used only when LP Note Style is set to Circuit and the selected Stochastic Layer is Note Prob.

Function Notes
GRID 1..16 Edit Note Variation Probability for the selected note target. Press 1..16 to set the Note Variation Probability level from low to high for the note target selected in 25..40.
If the selected target probability is 0, 1..16 stays OFF.
GRID 25..40 Note target select. 25..32 = semitone row.
33..40 = tone row.
32 = raise global Track octave by +1.
40 = lower global Track octave by -1.
If the chosen note exists in the current scale, note selection follows the scale.
Notes outside the current scale use a separate chromatic selection.
GRID 49..56 Octave select. -4 | -3 | -2 | -1 | 0 | +1 | +2 | +3
Selected octave is remembered.
GRID 57..59 Stochastic options. 57 = Use Loop | 58 = Clear Loop | 59 = Reseed
Reseed is blocked if Use Loop is enabled or the sequence is empty.
Double-press 25..40 Toggle selected note state. Toggle gate and store the current octave in the selected stochastic step.

Sequence Sub-View: Arp Circuit Keyboard

This sub-view is used only when LP Note Style is set to Circuit and the selected Arp Layer is Note.

Function Notes
GRID 1..16 Edit gate probability for the selected note target. Press 1..16 to set the gate probability level from low to high for the note target selected in 25..40.
If the selected target probability is 0, 1..16 stays OFF.
GRID 25..40 Note target select. 25..32 = semitone row.
33..40 = tone row.
32 = raise global Track octave by +1.
40 = lower global Track octave by -1.
Chooses a bypass-scale note target for the Arp Track.
GRID 49..56 Octave select. -4 | -3 | -2 | -1 | 0 | +1 | +2 | +3
Selected octave is remembered.
GRID 57..58 MIDI Keyboard options. 57 = Toggle MIDI Keyboard
58 = Toggle Hold
Hold works only when MIDI Keyboard mode is ON.
MIDI Keyboard mode Play live notes from the Keyboard area. Pressing 25..40 feeds the arpeggiator.
Releasing non-gated notes removes them.
Double-press 25..40 Toggle selected note state. When MIDI Keyboard mode is OFF, double-press toggles gate and adds or removes that note in the arpeggiator.
When MIDI Keyboard mode is ON, no double-press action is implemented.
TOP 5 + GRID Select Arpeggiator mode.

Pattern

Function Notes
TRK 1..8 Select the same Pattern on all Tracks. Launch scene-side Pattern 1..8 on all Tracks.
Launch timing follows the current execute type (Immediate | Sync | Latch).
GRID Select Track Pattern. Column: Track.
Row: Pattern.
Hold TOP 2..3 Latch | Sync execute types. Default execute type is Immediate.
Releasing TOP 2 commits latched requests.
If Pattern Change is set to Sync, TOP 3 temporarily flips back to Immediate.
TOP 1 + GRID Pattern Navigation. Shows the current Pattern Navigation map.
TOP 1 + TOP 2..5 Navigate Left | Right | Up | Down. Used on the current Pattern Navigation map.
TOP 8 + TRK 1..8 Track mute.
TOP 7 + TRK 1..8 Track Fill. Releasing TRK or TOP 7 clears Fills.
GRID colors Pattern state colors in the GRID. Colors vary with LP Style. They distinguish edited patterns, the currently playing pattern, and requested pattern changes.

Performer Overview
17
Toggle Performer follow
Track follow applies to the selected Track.
Sequence Length Temporary first/last step override across all Note | Curve | Stochastic | Logic Tracks. Hold one pad to loop that single step. Hold two pads to set a temporary first/last range. Releasing the held pad(s) restores the original range.
Overview GRID rows select Tracks and columns map visible steps. Toggles gates directly on Note, Stochastic (first 12 only), and Logic.
On Note Track rows, TOP 7 + GRID sets the pressed step to the scale base note or the same note at octave +5.
Current step, gates, and the octave +5 note use different LP Style color cues.
Function LEDs show the selected Track section as a bar.
