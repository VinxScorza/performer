PER|FORMER

Web Simulator

forked and maintained by Vinx Scorza

Current firmware version: v0.4.0

This browser simulator should be treated as a convenient reference build, not as a perfect representation of the hardware module or the desktop simulator. Timing, audio behavior, browser input handling, and some UI details may still differ from the real firmware experience on hardware or in the native simulator.

If the embed does not start correctly in your browser, open the standalone Web Simulator build: standalone Web Simulator build.
If you want to use real MIDI ports or a Launchpad controller, use the native Desktop Simulator instead of the browser build.

Sound Engine

In the browser simulator, a small sound engine is attached to the sequencer outputs so the generated gates and CV can be heard immediately.
The current build uses the refreshed Vinx demo state and the current Vinx simulator sound map:

  • TRK1: Kick
  • TRK2: Snare
  • TRK3: Clap
  • TRK4: Rim / percussive layer
  • TRK5: Closed hi-hat
  • TRK6: Open hi-hat
  • TRK7: Acid synth
  • TRK8: Curve lane controlling Run Mode
